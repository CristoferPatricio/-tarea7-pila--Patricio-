## 📋 Descripción del Proyecto

Este proyecto implementa un **sistema de bitácora** para la nave espacial USS Algoritmo utilizando el **TDA Pila (Stack)**. La bitácora registra todos los eventos de la misión en orden cronológico, permitiendo acceder al último evento registrado siguiendo el principio **LIFO (Last In, First Out)**.

### 🎯 Contexto de la Misión
Como Oficial de Sistemas de la USS Algoritmo, eres responsable de mantener un registro preciso de todos los eventos durante la misión. Cuando ocurre una emergencia, el capitán necesita ver inmediatamente el último evento registrado, no lo que pasó hace horas. Por eso, la pila es la estructura perfecta para este sistema.
