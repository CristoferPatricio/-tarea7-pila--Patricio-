# TDA Pila - Sistema de Bitácora de la USS Algoritmo

## Descripción
Sistema de bitácora para la nave espacial USS Algoritmo que implementa el TDA Pila (Stack) para registrar eventos siguiendo el principio LIFO (Last In, First Out).

## 📋 Requisitos Previos
- **Java Development Kit (JDK)** 8 o superior
- **Git** (opcional, para clonar el repositorio)
- **Terminal** (Command Prompt, PowerShell, o terminal de Linux/Mac)

## 🚀 Instrucciones para Compilar y Ejecutar

### Opción 1: Desde la terminal (Windows CMD)

```bash
# 1. Clonar el repositorio (si no lo tienes)
git clone https://github.com/CristoferPatricio/-tarea7-pila--Patricio-.git

# 2. Entrar a la carpeta del proyecto
cd -tarea7-pila--Patricio-

# 3. Compilar el archivo Java
javac src/Main.java

# 4. Ejecutar el programa
java -cp src Main
